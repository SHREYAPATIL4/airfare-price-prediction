# 📌 Project Overview

Airline ticket prices fluctuate due to several factors such as airline company, source city, destination city, duration, number of stops, and departure time. This project aims to build a Machine Learning regression model capable of predicting airfare prices using historical flight data.

The project demonstrates the complete Machine Learning pipeline including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and prediction.

---

# 🎯 Objectives

* Analyze airline ticket pricing data
* Perform data cleaning and preprocessing
* Handle categorical and numerical features
* Train a Machine Learning regression model
* Predict flight ticket prices accurately
* Evaluate model performance

---

# 📊 Dataset Information

The dataset contains flight-related information such as:

* Date of Journey
* Source

* Route
* Departure Time
* Arrival Time
* Duration
* Total Stops
* Additional Information
* Ticket Price (Target Variable)

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/airfare-price-prediction.git
```

Navigate to the project directory

```bash
cd airfare-price-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open:

```text
Airfare Price Prediction.ipynb
```

Run all notebook cells.
